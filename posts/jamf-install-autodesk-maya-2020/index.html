JAMF - Install Autodesk Maya 2020

The Admin
2021-07-17
MacOS, Autodesk, JAMF

In this case I’m using JAMF, and putting the extracting the files to the administrator desktop.

You will need to prestage the files on the computer. You can use Composer to create a DMG or PKG to extract the files on the endpoint.

sudo "/Users/administrator/prestage_files/Autodesk/InstallMaya2020.app/Contents/Helper/Setup.app/Contents/MacOS/Setup" -i deploy --silent -o "/Users/administrator/prestage_files/Autodesk/InstallMaya2020.app/Contents/Helper/manifest/deployment_maya.xml"

You can right Click on the package to show contents and edit the deployment_maya.xml, to inlcude your server info.

For more information see Basic Maya network deployment
